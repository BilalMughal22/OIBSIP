## Task 2 - Number Guessing Game

### Description
A Java game where the user guesses a random number between 1-100 generated by the computer.

### How it Works
1. A welcome dialog appears with Start Game and Exit options
2. Computer generates a random number between 1 and 100
3. User enters a guess through a dialog box
4. Computer responds with Higher or Lower hints
5. When user guesses correctly, attempts are counted and score is calculated
6. User can choose to play again or exit
7. Final score is displayed at the end

### Scoring System
- Guessed in 3 or less attempts will get you  10 points
- Guessed in 4 to 6 attempts will get you 5 points
- Guessed in more than 6 attempts will get you only 1 point

### Features
- Random number generation
- Higher/Lower hints
- Score system based on attempts
- Multiple rounds
- Input validation (empty input exception handled)

### How to Run
1. Compile: javac NumberGuessingGame.java
2. Run: java NumberGuessingGame

### Technologies Used
- Java
- JOptionPane (javax.swing)
